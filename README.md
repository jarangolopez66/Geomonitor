# 🗺️ GeoMonitor Vallejo

Plataforma participativa de mapeo geográfico para registrar y monitorear eventos en Vallejo, Córdoba, Colombia.

## 🎯 Características

- 📍 Registro de eventos (robos, inundaciones, basura, manholes)
- 🗺️ Mapa interactivo con Leaflet
- 🚁 Gestión de vuelos de drones
- 📊 Dashboard con análisis
- 🔐 Autenticación con GitHub
- ☁️ Almacenamiento en Firebase

## 🚀 Cómo usar

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. Inicia sesión con tu cuenta de GitHub
4. ¡Comienza a registrar eventos!

## 📋 Requisitos

- Navegador moderno
- Conexión a internet
- Cuenta de GitHub (para login)

## 🔧 Configuración Firebase

La app usa Firebase. Las credenciales están en el código (no es lo más seguro para producción).
